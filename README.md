# Library Management SQL Project

## About the Project
This project demonstrates the use of SQL to design and manage a library database. It focuses on creating tables and writing SQL queries to handle library operations.

## Database Tables
- Members
- Books
- Staff
- Issued Books

## Concepts Used
- CREATE TABLE
- Primary and Foreign Keys
- JOINs
- Aggregate Functions

## Tools Used
- MySQL
- SQL
